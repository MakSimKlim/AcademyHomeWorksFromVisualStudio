проверить в powerShell наличие git:

git config --global user.email "you@example.com" // регистрация в репозитории git
git config --global user.name "Rocket"           

cd D:\        // переход в папки репозитория
git status    // проверка на наличие изменений в репозитории git
git add -all  // добавление измененных файлов
git commit    // вход в коммит
git -h        // вызов help

в коммите:
написать коммит // любой текст
esc             // переход на нижнюю строку
shift+6         // для возможности ввода команд в нижней строке
:wq             // перезапись коммита (w) и  выход (q) из коммита

git status      // проверка статуса
